# if文
## 条件分岐
* ある条件が成り立つだけある処理を行う。
```
const number = 12;
```
//定数 number の値が１０より大きい

 console.log("numberは10より大きいです");

## 書き方
```
if (条件式) {
    処理 console.log~;
} セミコロンは不要
```
* コード例
const number = 12;

if (number>10){
 
    console.log("numberは10より大きいです");
}

* tabキーでインデント

## 条件式の出力
console.log(number>10);
```
true  ←条件が成り立つ
false ←条件が成り立たない
```
これらを 
## 真偽値
という 真偽値はこの二種類しかない
